# Contribuindo

Obrigado por querer melhorar o ZAIRIX.

## Como contribuir

1) Abra uma issue com:
- problema (bug) ou proposta (feature)
- contexto + passos para reproduzir (se bug)
- resultado esperado

2) Faça um PR com:
- descrição curta
- prints (se UI)
- riscos conhecidos

## Padrões de código

- Código simples e legível
- Evitar “mágica” sem comentário
- Preferir funções pequenas e puras

## Commits

- Mensagens curtas e objetivas (ex.: `fix: ...`, `feat: ...`, `chore: ...`)
