# Tarefas para Aplicação das Melhorias SignalHack

## 1. Refatorar Home (app/page.tsx)
- [ ] Substituir copy por promessas operacionais e tensionamento
- [ ] Estruturar fluxo mental: promessa > tensão > ganho rápido > remover risco > CTA único
- [ ] Remover explicações e seções descritivas
- [ ] CTA: "Quero meu sinal agora"
- [ ] Tipografia forte, espaçamento generoso, foco em escaneabilidade

## 2. Simplificar Wizard (/app)
- [ ] Reduzir para 3 passos objetivos
- [ ] Um campo por tela, exemplo visível, botão destacado
- [ ] Perguntas:
    1. O que você vê acontecendo no seu mercado? (exemplo visível)
    2. Quem pagaria para resolver isso? (exemplo visível)
    3. Como saber se funcionou? (exemplo visível)
- [ ] Barra de progresso simples (1/3, 2/3, 3/3)
- [ ] Microcopy: "Preencha para avançar", "Tudo certo, prossiga"
- [ ] Mensagens de erro amigáveis

## 3. Menu e Navegação
- [ ] Menu curto e orientado à ação: Detectar Demanda, Minhas Ofertas, Experimentos, Resultados, Plano, Suporte
- [ ] CTA dominante sempre visível

## 4. Visual e UX
- [ ] Títulos grandes, cor neutra, campos centralizados
- [ ] Exemplo em cinza claro acima do campo
- [ ] Botão CTA largo, cor única
- [ ] Feedback imediato ao preencher
- [ ] Espaço em branco generoso

## 5. Revisar Microcopy e Placeholders
- [ ] Botões: "Quero meu sinal agora", "Avançar", "Validar e Começar"
- [ ] Placeholders e exemplos claros em todos os campos
- [ ] Mensagens de erro: "Campo obrigatório", "Descreva o que vê", etc.

## 6. Testar e Validar
- [ ] Testar fluxo completo: Home > Wizard > Confirmação
- [ ] Garantir clareza em até 3 segundos
- [ ] Eliminar qualquer termo vago, acadêmico ou explicação longa

---

> Siga cada etapa, commitando por partes. Priorize Home e Wizard. Ajuste visual e microcopy ao final.
