# Sustentabilidade do Projeto

## Visão geral
Este projeto é mantido de forma independente por uma pessoa física. Não há investidores, empresas ou financiamento externo recorrente.

Todo apoio recebido é usado exclusivamente para custos operacionais, como hospedagem, domínio, infraestrutura e contas de e-mail.

## Como ajudar
- Doações diretas (Pix, PayPal, Stripe — se disponível)
- Apoio indireto via link de afiliado da Hostinger

## Transparência
- O projeto não exige contribuição para uso do software.
- Não coletamos dados adicionais para finalidades de monetização.
- Não há rastreamento de cliques sem consentimento explícito do usuário.

## Custos aproximados (exemplo)
- Hospedagem e Vercel/infra: R$ 50–200/mês (varia conforme uso)
- Domínio: R$ 15–50/ano
- E-mail e notificações: R$ 10–100/mês
